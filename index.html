Single-file site. Commit this file as index.html at repository root. Enable Pages.
# quick commands (run locally)
git init
git add index.html
git commit -m "Add portfolio"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git push -u origin main


# then enable GitHub Pages: Repository > Settings > Pages > Source: main / (root)
